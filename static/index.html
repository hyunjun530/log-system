🚀 로그 전송 API 가이드

POST /api/logs Header: X-API-Key: <your-key> Content-Type: application/json Body: { "level": "INFO", "message": { "timestamp": 1713682800000, "event": "user_action", "data": "click" } }

⏱ 타임스탬프 처리

- message 내부에 timestamp 필드가 있으면 해당 시간을 로그 시간으로 사용합니다.
- 지원 형식: Unix ms (숫자), RFC3339 문자열 등
- 필드가 없으면 서버 수신 시간이 자동 적용됩니다.

⚠️ 주요 실패 케이스

  • • 401 Unauthorized: X-API-Key가 누락되었거나 일치하지 않음
  • • 400 Bad Request: message가 JSON 객체({...})가 아님
  • • 400 Bad Request: 정의되지 않은 level (DEBUG/INFO/WARN/ERROR)
  • • 413 Payload Too Large: 단일 로그 크기가 64KB를 초과함
